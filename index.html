Как сделать так, чтобы файл скачивался:

Способ 1: Положить файл в ту же папку, что и HTML-файл, и указать его имя в атрибуте download тега <a>

Способ 2: Использовать JavaScript для создания временной ссылки с файлом (как в этом примере)

Способ 3: Загрузить файл на сервер и указать прямой URL к нему в атрибуте href
